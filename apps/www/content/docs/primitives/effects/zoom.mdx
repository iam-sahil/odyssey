## Usage

### Zoom

```tsx
<Zoom>Zoom</Zoom>
```

### Zooms

```tsx
<Zooms>
  <div>Zoom 1</div>
  <div>Zoom 2</div>
  <div>Zoom 3</div>
</Zooms>
```
