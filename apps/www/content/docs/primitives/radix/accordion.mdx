## Usage

```tsx
<Accordion>
  <AccordionItem>
    <AccordionHeader>
      <AccordionTrigger>Accordion Item 1</AccordionTrigger>
    </AccordionHeader>
    <AccordionContent>
      <div>Accordion Content 1</div>
    </AccordionContent>
  </AccordionItem>
  <AccordionItem>
    <AccordionHeader>
      <AccordionTrigger>Accordion Item 2</AccordionTrigger>
    </AccordionHeader>
    <AccordionContent>
      <div>Accordion Content 2</div>
    </AccordionContent>
  </AccordionItem>
</Accordion>
```
